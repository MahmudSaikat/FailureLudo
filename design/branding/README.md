# Branding Asset Paths (Professional Setup)

Use this structure for all logo/icon work so files stay organized and production-ready.

## 1) Source of Truth (editable design files)

- `design/branding/source/logo-master.svg`  
  Main vector logo (preferred file to share for edits).
- `design/branding/source/logo-mark-only.svg`  
  Icon-only variant (no text), optional but recommended.
- `design/branding/source/logo-wordmark.svg`  
  Text/logo lockup, optional.

## 2) Exported Assets for Google Play

- `design/branding/exports/play-store/app-icon-512.png`  
  Required for Play Console. Must be exactly 512x512 PNG.
- `design/branding/exports/play-store/feature-graphic-1024x500.png`  
  Optional Play listing graphic.

## 3) Exported Assets for Android Launcher Generation

- `design/branding/exports/android-launcher/ic-launcher-foreground-1024.png`  
  Transparent PNG foreground.
- `design/branding/exports/android-launcher/ic-launcher-monochrome-1024.png`  
  Monochrome icon for Android themed icons (optional but recommended).

## 4) Final Android App Resource Paths (generated by Android Studio)

Do not hand-edit these unless needed:

- `app/src/main/res/mipmap-anydpi-v26/ic_launcher.xml`
- `app/src/main/res/mipmap-anydpi-v26/ic_launcher_round.xml`
- `app/src/main/res/drawable/ic_launcher_foreground.xml`
- `app/src/main/res/drawable/ic_launcher_background.xml`
- `app/src/main/res/mipmap-mdpi/ic_launcher.webp`
- `app/src/main/res/mipmap-hdpi/ic_launcher.webp`
- `app/src/main/res/mipmap-xhdpi/ic_launcher.webp`
- `app/src/main/res/mipmap-xxhdpi/ic_launcher.webp`
- `app/src/main/res/mipmap-xxxhdpi/ic_launcher.webp`

## What to provide to Copilot for customization

At minimum, provide one of these files:

1. `design/branding/source/logo-master.svg` (best)
2. `design/branding/exports/android-launcher/ic-launcher-foreground-1024.png`

Also provide style notes in text:

- Color palette (hex values)
- Icon style (flat, gradient, minimal, bold)
- Whether text should appear in icon (usually no for launcher)
