# AI Work Log

This folder stores date-based logs for all AI-related work done in this repository.

## File Naming

- One file per day using local date:
  - `YYYY-MM-DD.md`

## Entry Format

```md
## HH:MM local
- Request: <what the user asked>
- Actions: <what was done>
- Files changed: <paths or "none">
- Commands run: <key commands or "none">
- Results: <metrics, pass/fail, key outcomes>
- Artifacts: <new/updated outputs or "none">
- Next steps: <suggested follow-up>
```

## Notes

- Append entries only. Do not rewrite history.
- Keep entries factual and concise.
- Use this log for reproducible AI engineering and research tracking.
